## What we did

## Decisions made
- Chose FastAPI over Django: faster, async-native, auto OpenAPI docs
- Chose PostgreSQL: most trusted open-source RDBMS, what companies use
- Chose monorepo structure: easier to manage as a 2-person team

## What we learned
- Git branching strategy: main → develop → feature/*
- Why conventional commits matter for readable history
- How to design a DB schema before writing code
